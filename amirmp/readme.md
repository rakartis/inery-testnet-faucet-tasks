### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```amirmp```

```shell
cd ./amirmp
```

Create .env and edit the variable
PRIVATE_KEY=YOUR PRIVATE KEY
URL_NODE=YOUR NODE URL

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
node ./rpc.mjs
```
