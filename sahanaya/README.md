### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- [Yarn](https://yarnpkg.com/)

### How to run

Change directory to ```sahanaya```

```shell
cd ./sahanaya
```

Open file `config.mjs` and set your information here:

```shell
vim config.mjs
```

Install dependencies

```shell
yarn install
```

Run the script

```
yarn inery-rpc
```
