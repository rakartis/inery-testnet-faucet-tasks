### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```nurulihsan```

```shell
cd ./nurulihsan
```

Create .env with varriables

```shell
nano .env
```

```
PRIVATE_KEY=YOUR PRIVATE KEY
INERY_ACCOUNT=YOUR INERY ACCOUNT
NODE_URL="http://YOUR_IP_NODE:8888"
```

Install dependencies

```shell
npm install
```

Run the script

Run Create

```
npm run create
```

Run Transfer

```
npm run transfer
```
