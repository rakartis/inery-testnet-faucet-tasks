**Requirements**
Python3


**Change Directory**

```
cd ./althio
```

**Install Requirements**

```
pip install -r requirements.txt
```

**Create .env and change its value**

```
cp .env-sample .env
```

**change env variable to match with your info**

```
PKEY=YOUR_PRIVATE_KEY
ACC=YOUR_INERY_ACC
```

**run the script**

```
python3 ./transaction.py
```
