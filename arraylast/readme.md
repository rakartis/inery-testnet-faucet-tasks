### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```arraylast```

```shell
cd ./arraylast
```

Create .env and edit the variable
PRIVATE_KEY=YOUR PRIVATE KEY
INERY_ACCOUNT=YOUR INERY ACCOUNT

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```

Run Create Token

```
npm run create
```

Run Issue Token

```
npm run issue
```

Run Transfer Token

```
npm run transfer
```
