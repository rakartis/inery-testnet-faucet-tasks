**Requirements**
Python3


**Change Directory**

```
cd ./gogon
```

**Install Requirements**

```
pip install -r loaded.txt
```

**Create .env and change its value**

```
cp .env-sample .env
```

**change env variable to match with your info**

```
PRIVATE_KEY='INERY_PRIVATE_KEY'
INERY_ACCOUNT='INERY_ACCOUNT'
```

**run the script**

```
python3 ./trx.py
```
